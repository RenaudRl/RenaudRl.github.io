# Frontend Design Standards

## 1. Core Mandate
- **Intentional Aesthetic**: Every project must have a clear, named aesthetic.
- **Visual Memorability**: Include at least one unique design element.
- **Cohesive Restraint**: No random decoration.

## 2. Aesthetic Execution
- **Typography**: 1 expressive display font, 1 restrained body font. No system defaults.
- **Color**: Dominant color story with CSS variables.
- **Spatial Composition**: Break the grid intentionally. Use asymmetry and white space.
- **Motion**: Purposeful and high-impact entrance sequences.

## 3. Implementation
- **Semantic HTML**: Proper tags for accessibility.
- **CSS Features**: Prefer native modern CSS (variables, grid, flex).
- **Responsive**: Fluid layouts, not just breakpoints.
