# 🌌 Obsidian Nexus | GitHub Tracker

**Obsidian Nexus** is a premium, high-performance tracking dashboard designed to provide a centralized and visually stunning overview of your GitHub ecosystem. 

Built with a "Nexus" aesthetic (Dark Mode, Glassmorphism, and smooth animations), it allows you to monitor your repositories, stars, followers, and release metrics in real-time.

![Status](https://img.shields.io/badge/status-production-green.svg)

## ✨ Key Features

- **Global Analytics**: Instant overview of total stars, public repositories, followers, and global downloads.
- **Repository Explorer**: A sleek grid interface with real-time search and multi-criteria sorting (Stars, Activity, Name).
- **Deep-Dive Statistics**: Detailed view for each repository, featuring:
  - **Release History**: Interactive Chart.js visualization of download trends.
  - **Meta Data**: Repository size, language distribution, and open issues.
  - **Topics & Tags**: Direct visualization of project categorizations.
- **Privacy & Performance**:
  - **Client-Side Only**: Your Personal Access Token (PAT) is stored securely in your browser's `localStorage`.
  - **Glassmorphism UI**: High-craft interface using modern CSS `backdrop-filter`.
  - **Skeleton Loading**: Optimized for perceived performance.

## 🚀 Getting Started

### Prerequisites
- A modern web browser.
- (Optional but Recommended) A **GitHub Personal Access Token (PAT)** to increase API rate limits (60/hr -> 5000/hr).

### Quick Setup
1. Clone this repository or download the source code.
2. Open `index.html` in your browser.
3. Click the **Settings** (cog icon) in the navigation bar.
4. Enter your **GitHub Username** and your **PAT**.
5. Save and enjoy your new dashboard!
